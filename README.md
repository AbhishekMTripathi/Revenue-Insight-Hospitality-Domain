# Revenue-Insight-Hospitality-Domain
🏨 Revenue Insight: Hospitality Domain
Project Overview
This project focuses on analyzing revenue data for a luxury hotel chain to identify performance trends, optimize pricing strategies, and improve overall occupancy rates. By leveraging Power BI and Excel, I transformed raw data into an interactive dashboard that provides actionable insights for stakeholders in the hospitality industry.

📊 Key Metrics (KPIs)
The analysis centers on the following industry-standard metrics:

RevPAR (Revenue Per Available Room): To measure the ability to fill rooms at a profitable rate.

ADR (Average Daily Rate): To understand the average rental income per occupied room.

Occupancy %: To track the utilization of available inventory.

SRN (Sellable Room Night) & URN (Utilized Room Night): To monitor capacity versus actual sales.

Realization %: To identify the ratio of successful check-ins against total bookings.

🛠️ Tech Stack & Tools
Power BI: Data modeling, DAX measures, and interactive visualization.

Excel: Initial data exploration and validation.

Power Query: ETL (Extract, Transform, Load) processes to clean and shape the data.

📂 Dataset Description
The project utilizes five main datasets:

dim_date: Calendar dimensions for time-series analysis.

dim_hotels: Information about hotel properties and categories.

dim_rooms: Details regarding room types (Standard, Elite, Premium, Presidential).

fact_bookings: Transactional data for individual bookings.

fact_aggregated_bookings: Summary data for daily room availability and utilization.

💡 Business Insights
Weekend vs. Weekday Performance: Identified significant revenue gaps between weekends and weekdays, suggesting a need for dynamic pricing.

Booking Channel Analysis: Analyzed which platforms (direct, OTA, etc.) drive the most revenue and have the highest cancellation rates.

City-wise Performance: Evaluated property performance across different cities to identify underperforming regions.
